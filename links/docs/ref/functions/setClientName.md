# Function: setClientName()

> **setClientName**(`client`): `void`

Call when the SPA starts, providing the name of the client. It should be the same as the subdomain, or "root" if it's the root domain.

## Parameters

| Parameter | Type     |
| --------- | -------- |
| `client`  | `string` |
